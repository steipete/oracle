# v18 Conformance Fixture Provenance

Every fixture in this harness is sourced from the v18 plan bundle that
ships alongside this repo. Fixtures are NOT regenerated by these tests;
they are treated as the canonical reference output.

## Source

- Bundle: `PLAN/oracle-vnext-plan-bundle-v18.0.0/`
- Bundle version: `v18.0.0` (see `PLAN/oracle-vnext-plan-bundle-v18.0.0/VERSION`)
- Schema: `PLAN/oracle-vnext-plan-bundle-v18.0.0/contracts/json-envelope.schema.json`
- Positive fixture: `PLAN/oracle-vnext-plan-bundle-v18.0.0/fixtures/json-envelope.ok.json`

## Regeneration

The plan bundle is regenerated upstream (see `scripts/contract-fixture-smoke.py`
in the bundle). Oracle does not regenerate it — when the bundle updates,
re-run this harness and update `COVERAGE.md` if the schema field set changes.
